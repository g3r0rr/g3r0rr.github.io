# MO Transfers4all Athens

Official website for **MO Transfers4all Athens** — premium private taxi and transfer services in Athens, Greece, operated by Marjus Oruci.

🌐 **Live site:** [mo-transfers4all.gr](https://mo-transfers4all.gr)

---

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main page — hero, booking form, fleet, prices, services |
| `privacy.html` | GDPR-compliant privacy policy |
| `terms.html` | Terms of service |
| `sitemap.xml` | XML sitemap for search engines |
| `robots.txt` | Crawler instructions |

---

## Features

- Bilingual — English & Greek with persistent language switcher
- Booking form with silent WhatsApp delivery via Sendzen API
- Address autocomplete powered by Geoapify
- GDPR compliant — immediate data deletion policy
- Cookie consent banner
- Floating WhatsApp contact button
- Transparent pricing table with night rate indicators
- Schema.org TaxiService structured data for SEO
- Open Graph meta tags for social sharing
- Mobile-first responsive design
- Scroll reveal animations

---

## Tech Stack

- Pure HTML / CSS / JavaScript — no frameworks
- [Sendzen](https://sendzen.io) — WhatsApp Business API
- [Geoapify](https://geoapify.com) — Address autocomplete
- [Google Fonts](https://fonts.google.com) — Cormorant Garamond & Montserrat
- Hosted on GitHub Pages
